# Share to Clipboard
Android app that adds "Copy" to Android share sheet. It also provide custom/partial copy options.

## Implemented
 * Copy shared text
 * Configurable behavior
 * Extract Numbers with different formats
 * Extract Links with different formats
 * Extract Emails
 * Just copy option
